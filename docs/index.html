üìã Project Details

Data Source

Conference location data sourced from the ASIH Past Meetings page.

Technical Stack

  • R - Data processing and statistical analysis
  • Shiny - Interactive web application framework
  • Leaflet - Interactive mapping library
  • ggplot2 - Static visualizations with viridis color palette
  • sf & maps - Geographic data processing
  • tidygeocoder - Geocoding city names to coordinates
  • GitHub Actions - Automated deployment pipeline

Key Features

  • Interactive dashboard with multiple visualization tabs
  • Year range and country filtering capabilities
  • Clickable map points with detailed meeting information
  • Summary statistics and data tables
  • Responsive design for desktop and mobile devices
  • Chronological progression visualization
